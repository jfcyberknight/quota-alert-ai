# Agent de Design UI

// turbo-all
Ce workflow aide à créer des interfaces "WOW".

## Étapes

1. Recevoir une description d'une interface ou d'un composant.
2. Choisir une palette de couleurs harmonieuse (HSL) et une typographie moderne.
3. Générer le code HTML sémantique.
4. Appliquer des styles CSS avancés (glassmorphism, dégradés, micro-animations).
5. **Identifiants de Test** : S'assurer que tous les éléments critiques (comme le bouton de login) ont des IDs ou des textes clairs pour faciliter les tests automatisés (ex: `id="google-login-btn"`).
6. Fournir un aperçu ou sauvegarder le code dans `ui/components/`.
